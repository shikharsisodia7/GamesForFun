GamesForGithub

This is a simple Python command-line project that combines three different programs into one file: a calculator, Tic Tac Toe, and Hangman. The goal of this project was to practice Python basics like functions, loops, conditionals, user input, game logic, and working with a menu-based program.

Features

1. Calculator
- Supports basic math operations like addition, subtraction, multiplication, division, modulus, floor division, and powers.
- Includes a scientific mode with math functions and constants like `pi` and `e`.
- Uses safer expression evaluation instead of directly running user input.

2. Tic Tac Toe
- Play a normal 2-player Tic Tac Toe game.
- Play against an AI.
- Includes different AI difficulty levels: easy, medium, and hard.
- Hard mode uses minimax logic for stronger decision-making.

3. Hangman
- Play with a random word chosen by the program.
- Option to enter a hidden word for another player to guess.
- Includes a hint option that reveals a letter but costs one life.
- Shows the Hangman drawing as the player makes wrong guesses.

How to Run

Make sure Python is installed on your computer. This project works with Python 3.8 or newer.
